# Changelog

## [0.1.0] - 2026-04-08

### Added
- Initial release
- P/Invoke bindings for libpiper_plus C API
- Synchronous TTS API (`PiperTTS`)
- Asynchronous TTS API (`PiperTTSAsync`) with CancellationToken support
- `AudioClipExtensions` for PCM-to-AudioClip conversion
- `PiperModel` ScriptableObject for editor workflow
- Custom Inspector with test synthesis button
- Model file importer for .onnx files
- Platform plugin placement guide
- Basic TTS demo sample
