## 2. 実装内容の詳細

### 対象ファイル

### 追加セクション

#### importmap サンプル (バンドラー不要)

```html
<!DOCTYPE html>
<html>
<head>
  <script type="importmap">
  {
    "imports": {
      "piper-plus": "https://esm.sh/piper-plus@0.3.1",
      "onnxruntime-web": "https://esm.sh/onnxruntime-web@1.24.3"
    }
  }
  </script>
</head>
<body>
  <button id="speak">Speak</button>
  <script type="module">
    import { PiperPlus } from "piper-plus";

    document.getElementById("speak").addEventListener("click", async () => {
      const piper = await PiperPlus.initialize("tsukuyomi");
      const audio = await piper.synthesize("こんにちは、世界！");
      audio.play();
    });
  </script>
</body>
</html>
```

#### WASM ビルドバリアント案内

| バリアント | WASM サイズ | 対応言語 | 用途 |
|-----------|------------|---------|------|
| `multilingual` (デフォルト) | ~X MB | JA/EN/ZH/KO/ES/FR/PT/SV | 全言語対応が必要な場合 |
| `ja` | ~X MB | JA のみ | 日本語専用アプリ |
| `ja-lite` | ~X MB | JA (軽量版) | モバイル・低帯域環境 |

各バリアントのインポート方法:
```js
// デフォルト (multilingual)
import { PiperPlus } from "piper-plus";

// 日本語専用
import { PiperPlus } from "piper-plus/ja";
```

## 5. 懸念事項とレビュー項目

### 懸念事項
- esm.sh の CDN 安定性。esm.sh がダウンした場合、サンプルが動作しなくなる。代替として unpkg.com や jsdelivr.net を併記すべきか
- importmap のブラウザサポート: Chrome 89+, Firefox 108+, Safari 16.4+ で対応。IE11 / 古い Safari では動作しない旨を注記する
- WASM バリアントのサイズは ビルド時点の値を記載する必要がある。CI で自動更新するか、手動で更新するか
- `onnxruntime-web` のバージョン固定。esm.sh の URL にバージョンを含めるため、onnxruntime-web のメジャーバージョンアップ時にサンプルが壊れる可能性がある
