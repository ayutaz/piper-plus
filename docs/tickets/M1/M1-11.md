## 2. 実装内容の詳細

### 通知内容

GitHub Issue コメントまたは Discussion で以下を伝える:

```markdown
Hi @endo5501,

We've recently added a C API shared library (`libpiper_plus`) and a Dart FFI example
to piper-plus. These might be of interest to you:

- **C API Header**: `src/cpp/piper_plus.h`
  - Opaque handle pattern, streaming callback, custom dictionary support
  - Phoneme timing API
- **Dart FFI Example**: `examples/dart/`
  - Demonstrates basic TTS synthesis via Dart FFI bindings
- **Godot GDExtension Example**: `examples/godot/`
- **Release CI**: Pre-built shared libraries for Linux/macOS/Windows
  (`.github/workflows/release-shared-lib.yml`)

We'd love to hear your feedback or if you have any use cases in mind.
Feel free to open an issue or discussion!
```
