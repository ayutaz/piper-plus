### タグ push

```bash
# dev ブランチの HEAD にタグを作成
git tag -a v1.12.0 -m "Release v1.12.0 - M1 milestone"
git push origin v1.12.0
```

### GitHub Release 作成

```bash
gh release create v1.12.0 \
  --title "v1.12.0" \
  --notes-file RELEASE_NOTES.md \
  --target dev
```

リリースノートには以下を含める:
- M1 マイルストーンの概要 (オンボーディング改善、エコシステム接続、モデル配布強化)
- 主要な変更点のサマリー (CHANGELOG の内容をユーザー向けに整理)
- アップグレード手順 (該当する場合)
- 既知の問題 (該当する場合)
