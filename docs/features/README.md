## Feature Overview

Piper TTS offers a comprehensive set of features for high-quality text-to-speech synthesis:

1. **Multiple Language Support** - Over 30 languages including Japanese
2. **Voice Customization** - Multiple voices per language
3. **Performance Options** - CPU and GPU acceleration
4. **Flexible Input** - Text files, phonemes, or direct text
5. **Output Control** - WAV files, streaming, or direct playback
