## 技術アーキテクチャ

```
日本語テキスト
    ↓
[MeCab WebAssembly]
    ↓ 形態素解析（~60ms）
[OpenJTalk WebAssembly]  
    ↓ 音素変換（~30ms）
[ONNX Runtime Web]
    ↓ 音声合成（モデル依存）
[Web Audio API]
    ↓ 
音声出力
```
