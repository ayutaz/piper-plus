## タスク管理ガイドライン

### 各タスクの構成
- **所要時間**: 最大2日（16時間）
- **依存関係**: 明確に定義
- **成果物**: 具体的で測定可能
- **完了基準**: 明確に設定

### 進捗管理
- 日次スタンドアップ（15分）
- 週次レビュー（1時間）
- タスク完了時に成果物確認
- 遅延時は即座にエスカレーション

### リスク対応
- 各タスクに20%のバッファ
- ブロッカー発生時は24時間以内に対応策決定
- 並行作業可能なタスクは積極的に並列化
