# MeCab WASM実装 - テスト手順まとめ

## 現在の状況

DARTSトライの実装で、ルートノードからの遷移に問題があることが判明しました。
- ルートノード: base=-2, check=199
- 標準DARTS公式: t = base + byte + 1
- しかし、UTF-8日本語文字（0xe3-0xe7）の正しい遷移パスが見つかっていません

## 実装した対策

1. **標準DARTS公式のみを使用**
   - 特殊なマッピングを削除し、シンプルな実装に戻しました
   - 辞書検索で見つからない文字は未知語処理でカバー

2. **未知語処理の実装**
   - 漢字、ひらがな、カタカナ、英数字、記号を適切に分類
   - 各文字種に応じた品詞情報を付与
