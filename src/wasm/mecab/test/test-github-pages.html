Unity WebGL 統合ガイド

Unity WebGLでの使用方法:

  1. ビルド設定 
    Player Settings > Publishing Settings > Compression Format: Gzip
Player Settings > Publishing Settings > Decompression Fallback: ON
  2. WebGLテンプレートでこのHTMLを参照
  3. jslibでMeCab関数をエクスポート: 
    mergeInto(LibraryManager.library, {
    MeCabParse: function(textPtr) {
        const text = UTF8ToString(textPtr);
        const result = window.mecab.parse(text);
        return allocateUTF8(result);
    }
});
  4. C#から呼び出し: 
    [DllImport("__Internal")]
private static extern string MeCabParse(string text);

GitHub Pages デプロイ手順

  1. このディレクトリをGitHubリポジトリにプッシュ
  2. Settings → Pages → Source: Deploy from a branch
  3. 辞書ファイルは dict/naist-jdic/ に配置
  4. gzip圧縮版（.gz）を用意すると転送量を削減可能
