🎤 OpenJTalk-Piper WebAssembly TTS デモ

ブラウザで動作する日本語音声合成システムのデモンストレーション

  • 標準版デモ

    OpenJTalkとPiper ONNXを組み合わせた基本的な音声合成デモです。安定した動作を提供します。

    標準版を開く

  • 最適化版デモ NEW

    並列ダウンロードとモデルウォームアップにより、初期化時間を大幅に短縮した高速版です。

    最適化版を開く BETA

  • 遅延読み込み版デモ NEW

    ONNXモデルを必要になるまで読み込まず、初期表示を高速化。Service Workerによるキャッシュ機能付き。

    遅延読み込み版を開く BETA

技術仕様

  • 音素化エンジン: OpenJTalk (WebAssembly)
  • 音声合成モデル: Piper VITS (ONNX Runtime Web)
  • 辞書: NAIST Japanese Dictionary
  • 音声: HTS Voice "Mei (Normal)"
