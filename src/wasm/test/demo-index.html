🎙️ Piper WebAssembly TTS デモ

動作要件:
  • Chrome 113以降（推奨）
  • WebAssembly サポート
  • 十分なメモリ（最低1GB RAM推奨）

利用可能なデモ

  • 完全統合TTSデモ
    MeCab → OpenJTalk → ONNX Runtime による完全な音声合成パイプライン
  • ストリーミングTTSデモ
    リアルタイムストリーミング音声合成（長文対応）
  • 実モデル音声合成デモ
    実際のPiperモデルを使用した高品質音声合成
  • パフォーマンスベンチマーク
    各処理段階の詳細なパフォーマンス測定とSIMD/WebGL最適化の効果測定
⚠️ 注意事項:
  • 初回ロード時に大きなファイル（辞書、モデル）のダウンロードが発生します
  • モバイルデバイスでは動作しない可能性があります
  • 実モデルデモには追加のONNXモデルファイルが必要です

技術スタック

  • MeCab: 日本語形態素解析エンジン（WebAssembly版）
  • OpenJTalk: 日本語音素変換エンジン（WebAssembly版）
  • ONNX Runtime Web: ニューラルネットワーク推論エンジン
  • Web Audio API: ブラウザネイティブ音声再生

パフォーマンス

典型的な処理時間（100文字の日本語テキスト）:

  • 形態素解析: ~60ms
  • 音素変換: ~30ms
  • 音声合成: モデル依存（通常100-500ms）
  • 合計: 200-600ms
